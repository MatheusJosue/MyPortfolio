# 🚀 Portfólio - Matheus Cerqueira Silva

Um portfólio moderno e responsivo de um Frontend Developer Sênior, construído com **Next.js 15**, **React 19**, **TypeScript** e **Tailwind CSS**.

## ✨ Características

- **Design Moderno**: Paleta de cores profissional com azul
- **Dark/Light Mode**: Toggle de tema com persistência em localStorage
- **Componentes Reutilizáveis**: Arquitetura limpa e desacoplada
- **Responsividade**: Mobile-first design para todos os dispositivos
- **Animações Suaves**: Framer Motion para transições elegantes
- **Performance Otimizada**: SSR, lazy loading e code splitting
- **SEO Completo**: Metadata dinâmica e Open Graph
- **Acessibilidade**: WCAG 2.1 AA compliant

## 🛠️ Stack Tecnológico

### Frontend

- **Next.js 15** - React framework com App Router
- **React 19** - Library UI moderna
- **TypeScript** - Type safety
- **Tailwind CSS** - Styling moderno
- **React Bootstrap** - UI components
- **Framer Motion** - Animações
- **React Icons** - Icon library
- **CSS Modules** - Scoped styling

### Desenvolvimento

- **Turbopack** - Build tool rápido
- **ESLint** - Code quality
- **Node.js + npm** - Runtime e package manager

## 🚀 Como Começar

### Pré-requisitos

- Node.js 18+ instalado
- npm ou yarn

### Instalação

1. **Acesse o diretório do projeto**

   ```bash
   cd my-portfolio
   ```

2. **Instale as dependências**

   ```bash
   npm install
   ```

3. **Inicie o servidor de desenvolvimento**

   ```bash
   npm run dev
   ```

4. **Abra no navegador**
   ```
   http://localhost:3000
   ```

## 📝 Scripts Disponíveis

```bash
npm run dev          # Inicia servidor dev com Turbopack
npm run build        # Compila para produção
npm run start        # Inicia servidor de produção
npm run lint         # Executa ESLint
```
